# React Frontend for File Management

A React application using Vite and React Router for managing file uploads and browsing files by category.

---

## Requirements

- Node.js (v22 or newer recommended)  
- npm or yarn  

---

## Setup & Installation

1. Clone the repository:

```bash
git clone https://your-frontend-repo-url.git
cd your-frontend-folder
```

2. Install dependencies

```bash
npm install
# or
yarn install
```

3. Create a .env file to customize environment variables (e.g., API base URL).

```bash
cp .env.example .env
```

4. Start the development server:
```bash
npm run dev
# or
yarn dev
```
